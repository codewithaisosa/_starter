# 🚀 Starter HTML/CSS/JS Template

[GitHub Repository](https://github.com/codewithaisosa/_starter)

A simple, lightweight, and beginner-friendly starter template for building websites using **HTML, CSS, and JavaScript**. This project is designed to help you skip setup time and jump straight into building your ideas.

---

## ✨ Features

* Clean and minimal project structure
* Organized HTML, CSS, and JavaScript files
* Easy to customize and extend
* Mobile-friendly starting point
* No frameworks or dependencies — pure web basics
* Great for learning or quick prototyping

---

## 📁 Project Structure

```
_starter/
│
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── assets/
    └── (images, icons, etc.)
```

---

## ⚡ Getting Started

### Option 1: Use this template

Click **“Use this template”** on GitHub to generate your own repository.

### Option 2: Clone the repository

```bash
git clone https://github.com/codewithaisosa/_starter.git
```

Then open the project folder:

```bash
cd _starter
```

Finally, open `index.html` in your browser.

---

## 🛠️ Customization

You can easily modify this starter to fit your needs:

* Edit `index.html` to change page content
* Update styles in `css/style.css`
* Add interactivity in `js/script.js`
* Replace assets inside the `assets/` folder
